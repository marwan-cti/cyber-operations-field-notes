# Prompt injection

## Status

Structured stub — to be expanded with sources.

## Why this topic matters

Prompt injection is a core AI Security issue because it targets the instruction-following behavior of language models and becomes more serious when models can use tools, retrieve data or interact with external systems.

## Analytical angle

- Direct vs indirect prompt injection.
- Tool use and agentic workflows.
- Data exfiltration risk.
- Security boundary confusion.
- Detection and logging challenges.

## Defensive relevance

This topic is relevant for AppSec (Application Security), SOC (Security Operations Center), AI Security and CTI teams monitoring emerging abuse patterns.
